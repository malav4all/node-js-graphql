> NestJS GraphQL code-first based initial project with [TypeGraphQL](https://typegraphql.ml). Read more [here](https://docs.nestjs.com/graphql/quick-start#code-first).

## Usage

```sh
> npm install
> npm start
```

## Tech Stack

- TypeScript
- Node
- Nest
- GraphQL
- Apollo
- TypeGraphQL
- PostgreSQL
- TypeORM
